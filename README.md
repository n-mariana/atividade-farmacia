# Sistema de Estoque - Farmácia VAV

Projeto CRUD desenvolvido em PHP com PDO e MySQL para gerenciamento de estoque de uma farmácia.

---

### 🚀 Tecnologias
- PHP
- MySQL
- PDO
- HTML5
- CSS3
- Git/GitHub
- XAMPP

---

### 📂 Estrutura do Projeto

```bash
farmacia-vav/
│
├── config/
│   └── conexao.php
│
├── includes/
│   ├── header.php
│   └── footer.php
│
├── css/
│   └── style.css
│
├── index.php
├── cadastro.php
├── editar.php
├── excluir.php
│
├── database.sql
└── README.md
```

---

### 🛠️ Funcionalidades
- Cadastro de produtos
- Listagem de produtos
- Edição de produtos
- Exclusão de produtos
- Layout responsivo

---

### 🗄️ Banco de Dados

Tabela `produtos`:

- id
- nome
- fabricante
- preco
- estoque

---

### 🎨 Design
- Mobile First
- Responsivo
- Regra de cores 60-30-10

---

### ⚙️ Como Executar

1. Coloque a pasta em:

```bash
xampp/htdocs/
```

2. Inicie Apache e MySQL no XAMPP

3. Importe o arquivo:

```bash
database.sql
```

4. Acesse:

```bash
http://localhost/farmacia-vav
```
